# itgc-checklist
A simple Flask web app that allows IT auditors to assess and report on key IT General Controls (ITGC) such as access management, change management, and backup policies.

## Features
- Select compliance status for each control
- Add auditor comments
- Auto-generate summary report
- Built with Python (Flask) + HTML

## How to Run
1. Install dependencies: `pip install flask`
2. Run the app: `python app.py`
3. Open browser at: `http://127.0.0.1:5000`
